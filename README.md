# sales-data-analysis 

# Project Overview
This project performs exploratory data analysis (EDA) on a sales dataset to uncover business insights such as revenue trends, product performance, and regional sales distribution. The analysis is done using Python and focuses on understanding key sales metrics. 

# Dataset Information
File Name: sales_large.csv
Format: CSV
Rows: 300
Columns: 6
Type: Sales Data 

# Features (Columns)
Column Name 	Description
OrderID ->	Unique identifier for each order
Product	-> Name of the product sold
Category -> 	Product category (e.g., Electronics, Furniture)
Sales	-> Sales amount per order
Quantity	-> Number of items sold per order
Region	-> Geographic region of the sale

# Objectives
The main goals of this project are:
Analyze total and average sales performance
Identify top-performing products and categories
Understand regional sales distribution
Explore quantity vs sales relationship
Perform basic business intelligence using Python

# Key Analyses Performed
✔ Sales Analysis
Total Sales
Average Sales
Maximum & Minimum Sales
✔ Product & Category Insights
Top-selling products
Category-wise performance
Sales contribution by category
✔ Regional Analysis
Sales distribution by region
Region-wise performance comparison
✔ Quantity Impact
Relationship between quantity and sales
